What is a rapplication?

A rapplication (RAPP application) is a converged single-file agent.py — one self-contained Python file that contains an entire AI workflow. Drop it in your local brainstem's agents/ directory and it works. No pip install. No native dependencies. Just BasicAgent and an LLM key in your environment.

Each rapplication was produced by the double-jump loop: claude vs framework, score head-to-head, encode the diagnosis as agent.py changes, re-run until convergence. Then collapse the multi-file ensemble into one shippable singleton.

Install path: 
curl -fsSL <singleton_url> > ~/your-brainstem/agents/<name>.py
# now POST /api/swarm/deploy with a bundle that includes it
# OR if your brainstem watches agents/, just hit the endpoint
