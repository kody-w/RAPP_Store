Talk to the catalog

This is the resident vBrainstem for the RAPP Store. Pick a rapplication from the sidebar, load it, and chat with it. Pure browser — your code, your token, your conversation.

1Sign in with GitHub. Top-right pill. Uses your GitHub account to call GitHub Models for the LLM (free tier, gpt-4o-mini default).
2Pick a rapp. Sidebar lists every rapplication in the catalog. Click one and it loads into the browser — its UI iframe mounts, its agent's perform() registers as a tool the model can call.
3Chat. The model has the loaded rapp's perform() as a tool — it calls into Pyodide to run real Python in the browser. Your code, your token, your conversation; nothing routes through a server.
4Take it home, bring it back. Each rapp card has an ↓ export button that bundles the singleton (+ service / UI when present) into a <id>.rapplication.egg. POST it as {"egg_b64": "<base64>"} to /api/binder/import on your local brainstem, OR drag the .egg file back onto this page to re-load it into a fresh vBrainstem session.

vBrainstem settings

Sign in with GitHub to enable chat. The vBrainstem uses GitHub Models (which ships with your account) — no separate API key required.
