Talk to the catalog

This is the resident vBrainstem for kody-w/RAPP_Store. Pick a rapplication from the sidebar, load it, and chat with it. Pure browser — your code, your token, your conversation.

1Sign in with GitHub. Top-right pill. Uses your GitHub account to call GitHub Models for the LLM (free tier, gpt-4o-mini default). On sign-in, the StoreNavigator agent (hosted in RAR) auto-loads — ask it "what should I install for X?" and it'll search the catalog for you.
2Or tether to a local brainstem. Run kody-w/RAPP/rapp_brainstem (default port 7071) and click the tether pill to route chat through it. You get the operator's soul, memory, model preferences, and every locally installed agent — not just rapp_store ones.
3Chat. In cloud mode the model has the loaded rapp's perform() as a tool — it calls into Pyodide to run real Python in the browser. In tether mode the local brainstem owns the tool loop end-to-end.
